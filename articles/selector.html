Use Radio Input to Change Content

October 28, 2015

Keywords: HTML Tables, Radio input, JavaScript, HTML5

Statement

When you select the car model and click ok, you get the image of the car. All this must be done using tables.



Answer

Car Selector
Select a car model
Audi
BMW
Fiat

Solution

function loadCar() {
	if (document.getElementById("Audi").checked == true) {
		document.getElementById("audiTab").className = "visible";
		document.getElementById("bmwTab").className = "hidden";
		document.getElementById("fiatTab").className = "hidden";
	}
	if (document.getElementById("BMW").checked == true) {
		document.getElementById("audiTab").className = "hidden";
		document.getElementById("bmwTab").className = "visible";
		document.getElementById("fiatTab").className = "hidden";
	}
	if (document.getElementById("Fiat").checked == true) {
		document.getElementById("audiTab").className = "hidden";
		document.getElementById("bmwTab").className = "hidden";
		document.getElementById("fiatTab").className = "visible";
	}
}

Explanation

  • * The checked property sets or returns the checked state of a checkbox.
  • * The className property sets or returns the class name of an element.
