The given array is ["1", "6", "2", "8"].

The resulting number is: ?

Solution

var myArray = ["1", "6", "2", "8"];

function myFunction(a) {
	var arr = a.sort().reverse().toString().split(",").join("");
	var number = Number(arr);
	document.getElementById("arrayDemo").innerHTML = number;
}

Explanation

  • * The sort() method sorts the values as strings in alphabetical and ascending order.
  • * The reverse() method reverses the order of the elements in an array.
  • * The toString() method converts a number to a string.
  • * The split() method is used to split a string into an array of substrings, and returns the new array.
  • * The join() method joins the elements of an array into a string, and returns the string.
  • * The Number() method converts the string into a number.
