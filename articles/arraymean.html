The Mean of the Numbers in an Array

October 29, 2015

Keywords: Algorithm, Array, JavaScript, HTML5

Statement

Write an algorithm that calculates the mean of the numbers in an array.

Example:
For an Array = [1, 34, 23, 12, 6, 3]
you will get the mean: 13.166666666666666


Answer

For a given array: myArray = [4, 58, 23, 12, 6, 31],
the mean is: ?



Solution

		
function avgSum(){
   var myArray = [4, 58, 23, 12, 6, 31];
   var total = 0;
     for (var i = 0; i < myArray.length; i++) {
         total += myArray[i]
     }
    var avg = total/myArray.length;
    document.getElementById("arrMean").innerHTML = avg;
 }

Explanation

  • * The length property sets or returns the number of elements in an array.
