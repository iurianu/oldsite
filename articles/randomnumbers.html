An Algorithm to Create an Array of Random Numbers

October 29, 2015

Keywords: Array, JavaScript, HTML5

Statement

Write an algorithm that returns an array containing 50 random numbers.

Answer

Clicking the button will show an Array of 50 random numbers between 1 and 100 in a new Window Object

Push the button to return the array:

Solution

function arrayFifty(n) {
var arr = []
while (arr.length < n){
  var randomnumber = Math.ceil(Math.random()*100)
  var found = false;
  for (var i = 0; i < arr.length; i++){
    if (arr[i] == randomnumber){
    	found = true;
    	break;
    }
  }
  if (!found)arr[arr.length] = randomnumber;
}
    document.getElementById("arrP").innerHTML = arr;
}

Explanation

  • * The length property sets or returns the number of elements in an array.
  • * The ceil() method rounds a number UPWARDS to the nearest integer, and returns the result.
  • * The random() method returns a random number from 0 (inclusive) up to but not including 1 (exclusive).
  • * The return statement stops the execution of a function and returns a value from that function.
